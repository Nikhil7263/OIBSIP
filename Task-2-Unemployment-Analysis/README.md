# Task 2: Unemployment Analysis with Python

## Project Overview

This project is completed as part of the Oasis Infobyte Data Science Internship.

The main objective of this project is to analyze unemployment data using Python. The project focuses on understanding unemployment trends across different regions and areas using data analysis and visualization techniques.

## Dataset

The dataset contains unemployment-related information such as:

- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- VS Code
- GitHub

## Project Workflow

1. Imported required Python libraries
2. Loaded the unemployment dataset
3. Checked dataset information
4. Checked missing values
5. Cleaned column names
6. Converted the Date column into datetime format
7. Performed exploratory data analysis
8. Created visualizations
9. Analyzed unemployment trends
10. Wrote final insights and conclusion

## Visualizations Created

- Distribution of Estimated Unemployment Rate
- Average Unemployment Rate by Region
- Rural vs Urban Unemployment Comparison
- Unemployment Rate Over Time
- Labour Participation Rate by Region
- Estimated Employed People by Region
- Correlation Heatmap

## Key Insights

- Unemployment rate varies across different regions.
- Some regions have higher unemployment rates compared to others.
- Rural and urban areas show different unemployment patterns.
- Unemployment rate changes over time.
- The correlation heatmap shows relationships between unemployment rate, employment, and labour participation rate.

## Conclusion

In this project, unemployment data was analyzed using Python. The analysis helped to understand unemployment trends across regions and areas. Different visualizations were created to identify patterns in unemployment rate, employment, and labour participation rate.

This project improved my understanding of data analysis, data visualization, and real-world dataset handling using Python.
